# HactoberFest2023

Hacktoberfest is an annual worldwide event held during the month of October. The event encourages open source developers to contribute to repositories through pull requests (PR).
This is a month long event in ehich we have to create pull requests from which four pull requests should be merged in the original repository. 
This time on creating a successful Merge of your PR a tree is planted from your side and on participating and creating PR you get holopin badges which you can share on your linkedin and other platforms that you want.
